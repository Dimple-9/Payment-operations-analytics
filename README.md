# Payment Failure & Settlement Insights 

## Overview 
This project analyzes the end-to-end digital payment lifecycle and demonstrates operational monitoring required to ensure reliable payment processing. 

## Objectives 
  - Analyze transaction success and failure patterns
  - Monitor settlement timelines and processing performance
  - Identify reconciliation exceptions
  - Provide operational insights to improve payment reliability
  
 ## Process Flow 
  The project models the payment lifecycle from customer initiation through gateway routing, issuer bank authorization, settlement, and transaction verification. 
  
  ## Dashboard Insights
  The Power BI dashboard highlights:
  - Transaction success rates and failure trends
  - Root causes of payment failures
  - Settlement processing timelines
  - Gateway performance comparison
  - Transactions requiring reconciliation review
  
  ## Key Findings 
  - Majority of transactions processed successfully
  - Issuer declines were the primary failure cause
  - Gateway performance varied across providers
  - Settlement timelines were largely within expected ranges
